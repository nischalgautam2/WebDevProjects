# Space-Explorers
Space Explorers is a web application that fetches and displays data about every astronaut who is currently in space.

## APIs used
- [Open Notify](http://open-notify.org/)
- [Wikimedia REST API](https://en.wikipedia.org/api/rest_v1/)
